WorldEdit: Commands & Utility

Our server is comprised of many addons and data-packs. One of these is the Worldedit addon. World Edit is an awesome tool in Minecraft that lets you quickly change the world around you, by helping you alter and create structures a lot more easier than the traditional ways of playing, Utilising a vast array of commands. It is one of the most popular minecart mods/addons, and is truly a builders dream!

Basic Commands

//wand: This gives you a magic wand in the game. With it, you can select areas by clicking.

//pos1 and //pos2: These commands set the beginning and end points of your selection. You can also just click with your wand to set these points.

//set <block>: This one's like magic. It changes all the blocks in your selected area to whatever type you want. So, if you want to turn everything into stone, just type "//set stone".

//replace <from> <to>: This is super handy. It swaps one type of block for another in your selection. For example, if you want to turn all the dirt into grass, just type "//replace dirt grass".

//copy: It copies the blocks you've selected.

//cut: This is like copying, but it also removes the blocks from where they were.

//paste: After copying or cutting, this command lets you place those blocks back down.

//undo: Oops! Made a mistake? This command can undo your last action.

//redo: If you undid something by mistake, this brings it back.

//expand <direction> <amount> and //contract <direction> <amount>: These make your selection bigger or smaller in a particular direction. Very useful!

That's a good start! With these, you'll be able to do a lot in Minecraft with WorldEdit, however there are a lot of more sophisticated and complex command available, but these aren't necessary in order to make great use of this mod! If, nevertheless, you still wish to learn more you may continue reading this page. Here are some more sophisticated command of this mod.

 

More handy commands

//paste -a: This command pastes the copied region without air blocks, meaning only blocks with substance will be pasted. It's useful for copying and pasting structures without having to worry about filling in air blocks.

Example: //paste -a 

//generate: This command generates terrain in the selected region. It's handy for quickly creating landscapes or filling in areas with natural features like trees or ores.

Example: //generate trees (which will generate a landscape of, Trees. of course)

//smooth: This command smooths out terrain within the selected region. It's useful for making rough landscapes look more natural.

Example: //smooth 3 (number is the level of smoothness, practically) 

//hollow: This command hollows out a solid structure, leaving just the outer shell. It's helpful when you need to create a shell for a building or clear out the interior of a structure.

Example: //hollow

//cyl: This command creates a cylinder of blocks. It's useful for building towers, pillars, or other cylindrical structures.

Example: //cyl stone <radius> <height> 

//faces: This command fills in the faces of a selection, creating a solid block where previously only the edges existed.

Example: //faces 

//replacenear: This command replaces blocks near the player's position within a specified radius. It's helpful for making large-scale changes to the environment without having to select a specific region.

Example: //replacenear 10 stone grass (this specific command will replace all stone blocks within 10 blocks of the user with grass)

//regen: This command regenerates the blocks within the selection. It's useful for restoring areas that have been modified back to their original state.

Example: //regen

//overlay: This command overlays one material over another within the selection. It's helpful for adding textures or patterns to existing structures.
